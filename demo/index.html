How it works

Predictions are generated twice daily (11:05 and 13:05 UK time) by a GradientBoostingRegressor trained on ~8 years of UK electricity market data: wholesale prices, grid generation mix (gas / wind / nuclear / solar), wind speed forecasts at 9 UK offshore and onshore wind farms, and weather ensemble spread from ECMWF. The model uses a per-half-hour Kalman filter for online regime tracking and blends with the AgilePredict API baseline.

Validation: 5-fold GroupKFold by date, CV R² ≈ 0.88, MAE ≈ 1.90 p/kWh. See METHODOLOGY.md for the full approach, feature set, and validation results.

Data feed

Raw predictions are available as static JSON on Cloudflare R2, with no authentication, no rate limits, and CDN caching at Cloudflare's edge: 

curl https://data.gridahead.chrischung.nl/predictions/C.json

Endpoints:

  • /predictions/latest.json — all 14 regions, next 3 days
  • /predictions/{A..P}.json — per-region files (smaller)
  • /predictions/meta.json — forecast run metadata + model info

Full schema and example responses are in the repository README.
