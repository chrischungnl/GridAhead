# GridAhead

3-day ahead half-hourly UK electricity price predictions for Octopus Agile, covering all 14 DNO regions.

Predictions are updated twice daily and are available before Octopus publishes the official rates — giving you a head start on planning consumption, battery storage, and EV charging.

## Getting started

Request an API key by opening an issue on this repo.

```bash
curl -H "X-API-Key: YOUR_KEY" \
  "https://gridahead.fly.dev/api/v1/predictions?region=C"
```

## Endpoints

### `GET /api/v1/predictions`

Returns half-hourly price predictions in pence/kWh (inc. VAT).

| Parameter | Description | Default |
|-----------|-------------|---------|
| `region` | UK region code (A-P) | All regions |
| `date` | Single date YYYY-MM-DD | Tomorrow + 2 days |
| `date_from` | Start date | Tomorrow |
| `date_to` | End date | date_from + 2 days |

**Example response:**
```json
{
  "generated_at": "2026-04-09T12:00:00+00:00",
  "regions": {
    "C": {
      "code": "C",
      "name": "London / South East",
      "dates": {
        "2026-04-10": {
          "slots": [
            {
              "from": "2026-04-10T00:00:00+01:00",
              "to": "2026-04-10T00:30:00+01:00",
              "pence_inc_vat": 12.45,
              "low": 8.2,
              "high": 16.7
            }
          ],
          "summary": {
            "avg": 15.8,
            "min": 3.2,
            "max": 32.1,
            "count": 48
          }
        }
      }
    }
  }
}
```

Each slot includes:
- `pence_inc_vat` — predicted price
- `low` / `high` — confidence band (10th and 90th percentile)

## Authentication

Pass your API key via the `X-API-Key` header:

```bash
curl -H "X-API-Key: YOUR_KEY" "https://gridahead.fly.dev/api/v1/predictions"
```

Or as a query parameter: `?api_key=YOUR_KEY`

## Rate limits

- 30 requests per minute
- 1,000 requests per day

## How it works

Predictions are generated by a machine learning model trained on 8 years of UK electricity market data including:

- Historical wholesale prices
- National grid generation mix (gas, wind, nuclear, solar)
- Interconnector flows (France, Netherlands, Belgium, Norway, Denmark, Ireland)
- Day-ahead demand forecasts
- Wind generation forecasts from 9 UK offshore/onshore locations
- Weather ensemble uncertainty from ECMWF (51-member spread)

The model uses a stacking ensemble architecture with regime detection to handle different market conditions (normal, high-wind surplus, demand stress). Predictions include confidence bands from quantile regression.

Typical accuracy: ~1.7p/kWh mean absolute error on recent data.

## Use cases

- **Smart home automation** — schedule appliances for cheapest half-hours
- **Battery storage** — charge during predicted lows, discharge during highs
- **EV charging** — find the cheapest overnight slots before rates are published
- **Energy dashboards** — show users upcoming price forecasts
